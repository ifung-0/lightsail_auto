# Quick Start Guide

## Get Started in 3 Minutes

### Step 1: Install
```bash
# Run setup
python setup.py
```

### Step 2: Configure
Edit `config.json` with your LightSail login:
```json
{
  "username": "your_username",
  "password": "your_password"
}
```

### Step 3: Run
```bash
# FREE version (recommended)
python lightsail_bot_free_ai.py
```

That's it! The bot will:
1. Log into LightSail
2. Pick a book
3. Start reading
4. Answer questions automatically
5. Flip pages every 2 minutes

## Stop the Bot
Press `Ctrl+C` anytime to stop.

## View Progress
- Watch the terminal for updates
- Check screenshots of questions
- See session summary when stopped
